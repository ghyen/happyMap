{{유튜버 채널명}} 채널을 통해 공공임대주택 정보를 알기 쉽게 전달해 주시는 모습에 항상 감사드리며, 구독자로서 많은 도움을 받고 있습니다.

저는 **서울시 청년매입임대주택 공고 정보를 지도 위에서 한눈에 검색할 수 있는 웹 서비스**를 개발·운영하고 있는 {{본인 이름}}입니다. 공고가 올라올 때마다 수백 개의 매물을 PDF로 확인해야 하는 불편함을 해결하고자 이 서비스를 만들었습니다.

## 서비스 주요 기능

- **지도 기반 매물 탐색** — 서울시 전체 매입임대 매물을 지도 위에 표시하여 위치를 직관적으로 파악
- **다양한 필터링** — 자치구, 전용면적, 보증금, 월임대료, 방 개수, 승강기 유무로 조건에 맞는 매물만 필터링
- **출퇴근 시간 계산** — 회사 주소를 입력하면 각 매물에서 출퇴근 소요시간을 자동 계산하여 표시
- **보증금/월세 전환 시뮬레이션** — 전세전환·월세전환 비율을 조절하여 실제 부담 금액을 미리 확인
- **PDF 자동 파싱** — 새 공고 PDF를 업로드하면 데이터를 자동으로 추출하여 바로 검색 가능
- **모바일 지원** — PC와 모바일 모두에서 편리하게 사용 가능
